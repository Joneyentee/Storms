### Description

Interior courtyard of the royal palace, located behind the main rampart defenses. During Session 25, this area became an active battlefield as Falcon Faction forces breached the palace perimeter. Fighting between royal guards and orange/blue mercenaries spread throughout the courtyard, with flames visible from multiple directions as the palace siege intensified.

### Geography

Open courtyard space within palace walls, positioned between outer ramparts and main palace building. Multiple access points connect to palace corridors, throne room approaches, and defensive positions. Surrounded by palace architecture with windows overlooking courtyard. Stone paving throughout.

### Key Features

- **Multiple Access Routes** - Connects ramparts, corridors, throne room, and exterior defenses
- **Open Combat Space** - Allows for larger-scale melee engagements
- **Palace Windows** - Upper floors overlook courtyard (flames visible Session 25)
- **Defensive Chokepoints** - Various doorways and passages for tactical positioning
